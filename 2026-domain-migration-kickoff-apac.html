Key Reminders

APAC 팀 주요 안내사항

인벤토리 항목 노란색 표시: 작업 중인 항목은 반드시 노란색으로 표시. 글로벌 IT 팀이 중복 작업 방지를 위해 노란색 항목은 건드리지 않음.
Global VPN Migration Permission Group 필수 추가: 아직 Vantive 미이동 PC에는 Administrator 속성에서 Global Native Migration Permissions 그룹을 반드시 추가해야 글로벌 IT팀이 원격 접근 가능. 이미 Vantive로 이동된 PC는 추가 불가 (정상 완료 표시).
PowerShell 문제 발생 시 Net Framework 업그레이드: PowerShell 문제 발생하는 PC는 최소 .NET Framework 4.8 이상으로 업그레이드 필요.
WFTC 원격 디렉토리 임시 접근 방법: 현재 Vantive SailPoint 미생성 상태. Baxter SailPoint에 요청 제출 시 global\계정명 (백슬래시)으로 인증하면 서버 접근 가능.
로컬 IT 처리 불가 시 글로벌 팀 대행: 에러가 너무 많을 경우 로컬 팀은 작업을 중단하고 글로벌 팀에 요청. 다만 필요 애플리케이션 설치 및 Global VPN Permission Group 추가는 반드시 사전 완료 필요.
